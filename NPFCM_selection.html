Variable Selection in NPFCM

Rahul Ghosal and Arnab Maity

15 March 2020

Introduction

This is a demonstration of the variable selection method for NPFCM proposed in Ghosal and Maity (2020). The whole process is based on using the following steps.

  • Using the pprocess function to smooth noisy covariates.

  • Finally using the NPFCM.select function, which performs variable selection from the given input data.

All the functions mentioned above are included in the source file NPFCMselection.R.

Adding simulated covariates

set.seed(1)                # Set seed for reproducibility
p=16
A<-matrix(0,n,p)
B<-matrix(0,n,p)
for(i in 1:n)
{for(j in 1:p)
{
  A[i,j]<-rnorm(1,0,2) #mean 50 in parametric
}
}
for(i in 1:n)
{for(j in 1:p)
{
  B[i,j]<-rnorm(1,0,2) 
}
}
X<-function(i,j,t){A[i,j]*sqrt(2)*sin(2*pi*j*t)+B[i,j]*sqrt(2)*cos(2*pi*j*t)}

for(i in 1:2512)
  for(j in 7:22)
  {
    {mydata[i,j]<- X(mydata$id[i],(j-6),mydata$time[i])}}

Performing Variable Selection

#######Inputs############
# @ y= response                      
# @ mydata = a dataframe in long format column 1=id, column 2 = time,
# rest of the column covariates
# @ basist = Number of basis functions in t direction.
# @ basisx = Number of basis functions in x direction. 
######################### 
#Use larger grid in practice depending on the domain.
NPFCM.select(y,mydata,basist=c(4:6),basisx=c(4:6))    
## $NFSCAD
## [1] "atemp"     "hum"       "windspeed"
## 
## $NFMCP
## [1] "temp"      "hum"       "windspeed"

Both the NPFSCAD and NPFMCP for NPFCM selects only the weather variables and discards the pseudovariables.

  • NPFSCAD selects “atemp” “hum” “windspeed”
  • NPFMCP selects “temp” “hum” “windspeed”
